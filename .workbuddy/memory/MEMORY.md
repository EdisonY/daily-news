# Daily News 项目长期记忆

## 项目概述
每日新闻自动化抓取系统，包含四个模块：
1. GitHub 热门项目（10个）
2. 创业投资新闻（10条）
3. 小成本创业机会（10条）
4. 游戏设计灵感（10条，侧重玩法创新）

## 技术架构
- **触发方式**: GitHub Actions 定时触发（每日北京时间 08:00）
- **部署方式**: GitHub Pages
- **推送方式**: 企业微信机器人 Webhook
- **去重机制**: TF-IDF + SimHash + 历史记录

## 用户偏好
- **游戏设计偏好**: 玩法创新
- **推送语言**: 中文
- **推送时间**: 每日 08:00 北京时间
- **去重要求**: 排除相同新闻，避免重复推送

## 配置信息
- **GitHub Secrets**: 需要配置 `WECOM_WEBHOOK_URL`
- **去重阈值**: 标题相似度 0.7，内容汉明距离 3
- **历史记录保留**: 30 天
